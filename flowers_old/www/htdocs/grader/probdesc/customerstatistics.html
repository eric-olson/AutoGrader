Lab: CustomerStatistics

Specification

Vector<string> report_duplicates(Vector<string> names) { // you write code here }

Problem Statement

You will be given Vector<string> names, containing a list of customer names extracted from a database. Your task is to report the customers that occur more than once in this list, and the number of occurrences for each of the repeated customers.

Your function should return the report as a Vector<string>. Each element in this vector should be of the form "NAME OCCURS", where NAME is the name of one customer and OCCURS is the number of times his name occurs in names. Sort the result in alphabetical order by customer name.

Constraints

  • names contains between 1 and 50 elements, inclusive.

  • Each element of names contains between 1 and 50 characters, inclusive.

  • Each element of names contains uppercase letters ('A'-'Z') only.

Examples

  1. names = {"JOHN", "BOB", "JOHN", "BILL", "STANLEY", "JOHN"}

Returns: {"JOHN 3"}
    The only repeated name is JOHN, and it occurs three times. 

  2. names = {"YETI", "YETI", "YETI", "BIGFOOT", "BIGFOOT"}

Returns: {"BIGFOOT 2", "YETI 3" }
    Note the sorting order. 

  3. names = {"ANDREW", "BILL", "CINDY", "DOH", "ERGH", "FOO", "GOO", "HMPH"}

Returns: {}
    No repeated names this time. 

  4. names = {"THEONLYCUSTOMER"}

Returns: {}
    Again, no repeats. 

  5. names = {"A", "B", "A", "C", "A", "B", "A", "D", "D", "D"}

Returns: {"A 4", "B 2", "D 3"}
