Lab: Digit Sum

Function

   int digit_sum(int number) {
       // fill in code here
   }

Problem Statement

Given a number, return the sum of the digits in the number.

Write the function digit_sum that calculates and returns the sum of digits in its parameter number.

Constraints

  • number is greater than or equal to 0

Examples

  1.  81

 Returns: 9
    8 + 1 = 9 

  2.  1075

 Returns: 13



  3.  0

 Returns: 0



  4.  123456

 Returns: 21
