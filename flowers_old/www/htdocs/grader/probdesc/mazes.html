Problem Statement

In short, your job is to discover if there is a way to successfully solve a rectangular maze (see example below): 

########
#.##..*#
#.##.###
#.#..###
#.##...#
#o...#.#
########
In the maze, the start and end points are notated as 'o' and '*'. The '#' character represents a wall. Movement is up, right, down, or left; no diagonal moves allowed! The above maze is solvable; here is one which is not solvable: 
...#.*
.#..##
.#...#
o#....
Note that the only legal routes are those within the maze boundaries - even if there are no walls around the maze.

Create a function has_solution, which takes a vector of strings maze and returns true if the maze has a solution, and false otherwise.

Notes and Constraints

  • Mazes are allowed to contain looping paths; don't get lost looping forever!

  • The only legal moves are up, down, left, and right on squares marked with a period, the character 'o', or the character '*'.

  • There are between 1 and 16 rows in the maze, inclusive.

  • There are between 1 and 16 columns in the maze, inclusive.

  • The maze is perfectly rectangular. That is, each element of maze will be of the same length.

  • There are between 0 and (rows*columns-2) walls in the maze, inclusive.

  • There is exactly one 'o'.

  • There is exactly one '*'.

  • Each character in each element of maze can only be one of the following: a period (representing a space), a '#' (representing a wall), an 'o' (the starting point), or a '*' (the exit).

Examples

 "#o.*#",
