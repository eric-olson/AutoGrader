Lab: Count Appearances

   int number_times_appear(int number, int digit) {
       // fill in code here
   }

Problem Statement

Given a number and a digit, return the number of occurrences of the digit in the number.

Write the function number_times_appear that calculates and returns the number of occurrences of its parameter digit in its parameter number.

Constraints

  • number is greater than 0
  • 1 ≤ digit ≤ 9; i.e., digit is a nonzero digit 1 to 9

Examples

  1.  56854  5

Returns: 2
    The digit 5 appears twice. 

  2.  587673   4

 Returns: 0



  3. 12312312   2

Returns: 3



  4. 555555  5


Returns: 6
